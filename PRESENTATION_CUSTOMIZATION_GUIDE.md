# Presentation Customization Guide

Files created:

- `Large_Deal_Report_Automation_Presentation.pptx` (standard deck)
- `Large_Deal_Report_Automation_Presentation_BRANDED.pptx` (branded deck)

## Quick Customization Checklist

1. Open `Large_Deal_Report_Automation_Presentation_BRANDED.pptx`.
2. On slide 1, replace:
   - `Your Team Name`
   - `Your Name`
   - date text
3. Replace the `Add Team Logo` placeholder with your logo image.
4. Optional: adjust colors in PowerPoint using Design -> Variants -> Colors.

## Suggested Brand Mapping

- Primary brand color -> slide title and footer bar
- Secondary brand color -> section accents and highlights
- Neutral dark gray -> body text
- Light background -> code blocks and content panels

## Delivery Modes

- **10-minute version:** use all core slides and one code snippet slide.
- **7-minute version:** skip Q&A backup and one code proof slide.
- **15-minute workshop version:** keep all slides and spend more time on the worksheet activity.

## Speaker Notes Location

Speaker notes are embedded in the deck for each slide.

Use:

- Normal mode for editing content
- Presenter view to read notes while presenting
